**LIST OF COMMANDS**
-
**General commands**
```
  [$idols]         - List actual idol owners
  [+]              - Only on Idol Channel. Add a idol (ex '+Bob')
  [-]              - Only on Idol Channel. Remove a idol (ex '-Jack')
  [+Bob from Jack] - Only on Idol Channel. Add Bob, remove Jack
  [?]              - Only on Idol Channel. More info about an idol (ex '?Jack')
```
